OrangeExtensions

A pluggable jar containing stubs for the Apple Java Extensions, updated for Java 5 & 6.

Overview

Apple's official jar of Java extension stubs allows you to compile projects that use Apple Java Extensions on platforms other than Mac. Unfortunately, their jar file is for Java 4 only, and has not been updated for Apple's new extensions. I have updated their jar to the latest specification. Just drop it in your classpath and you're good to go!

Features

  • Allows you to compile Apple Java Extensions on Windows, Linux, and others.
  • Supports new methods like setDockImage() added for Java 5.
  • Supports little-known extensions like revealInFinder() added for Java 5 update 6 and Java 6 update 1.
  • Includes source code (unlike official jar), released under permissive BSD license.
