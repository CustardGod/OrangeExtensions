OrangeExtensions

A pluggable jar containing stubs for the Apple Java Extensions, updated for Java 5.

Overview

Apple's official jar of Java extension stubs allows you to compile projects that use Apple Java Extensions on platforms other than Mac. Unfortunately, their jar file is for Java 4 only, and has not been updated for new functions like setDockImage(). Also, Apple does not provide source code, so it can be difficult to combine with copyleft code. I have updated their jar to the latest specification and included source code. Just drop it in your classpath and you're good to go!
